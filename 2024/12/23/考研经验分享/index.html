2025考研408考研经验分享

2024/12/23 考研

考研内容

第一天 – 政治

  • 分值: 100
  • 学习内容: 马原 + 毛中特 + 新思想 + 史纲 + 思修 + 时政
  • 试卷题型(括号内是分值)
    • 选择题: 单选16(1) + 不定项17(2)
    • 分析题: 5个大题: 马原(10) + 新思想(10) + 史纲(10) + 思修(10) + 时政(10)

第一天 – 英语

  • 分值: 100
  • 内容: 考研单词6000
  • 英语二试卷题型(括号内是分值)
    • 选择题: 完型20(0.5) + 阅读5×4(2) + “新”题型5(2)
    • 非选择题: 短文翻译(15) + 应用文写作(10) + 图表作文写作(15)
  • 英语一试卷题型
    • 选择题: 完型20(0.5) + 阅读5×4(2) + “新”题型5(2)
    • 非选择题
    • 非选择题: 句子翻译5(5x2=10) + 应用文写作(10) + 图画作文写作(20)

第二天 – 数学

  • 分值: 150

  • 数学一内容: 高数(无经济学)86分+线代32分+概率论32分

    • 高数
      • 函数与极限
      • 数列与极限(难)
      • 一元微分学概念+计算+应用(极值拐点, 中值定理)+物理应用
      • 一元积分学概念+计算+应用(面积等, 中值定理)+物理应用
      • 多元函数微分学
      • 二重积分
      • 微分方程
      • 无穷级数
      • 多元函数积分学
      • 空间解析几何
    • 线代
      • 行列式
      • 矩阵和秩
      • 向量组
      • 线性方程组
      • 特征值和相似
      • 二次型
    • 概率论
      • 没学过不知道

  • 数学二内容: 高数(无经济学, 无级数, 无空间相关内容)118分 + 线代(无空间相关内容)32分

    • 高数
      • 函数与极限
      • 数列与极限
      • 一元微分学概念+计算+应用(极值拐点, 中值定理)+物理应用
      • 一元积分学概念+计算+应用(面积等, 中值定理)+物理应用
      • 多元函数微分学
      • 二重积分
      • 微分方程
    • 线代
      • 行列式
      • 矩阵和秩
      • 向量组
      • 线性方程组
      • 特征值和相似
      • 二次型

  • 数学三内容:

    • 你们不会考数学三的

第二天 - 408

  • 分值: 150
  • 40选择(2分) + 7综合题
    • 数据结构45分: 11选择 + 1算法题 + 1简答题
    • 计算机组成原理45分: 11 + 选择 + 2简答题
    • 操作系统40分: 11选择 + 2简答题
    • 计算机网络25分: 8选择 + 1简答题

复习流程(参考)

  • 时间上的安排只是理想状态下的, 会跟实际情况有些出入, 具体根据自己的情况来安排, 过度在意进度只会徒增焦虑, 只学了80%你就把那80%学好就行了, 你不需要考满分

408:

模拟时间安排

  • 12/01 - 12/21: 空着
    • 查缺补漏, 回顾错题等, 自行安排
  • 11/01 - 11/30: 真题
    • 2天一套
  • 10/01 - 10/31: 空着
    • 做回顾和整理, 查缺补漏
  • 08/01 - 9/30: 强化
    • 半个月一门
    • 重点在大题
  • 2/01 - 7/31: 一轮(4门课一轮: 1.5月/门)
    • 一轮一般只做选择题, 综合题如果有特殊情况可以自己选择性做一下
    • 注意最好做一下适当的总结, 后面肯定会忘很多, 所以要做点措施让自己少汪一点

学习内容推荐(非常主观)

  • 建议尽量减少视频课时长, 就算内容很全, 看完了你也记不住
  • 王道的课有点墨迹
  • 一轮
    • 数据结构: b站 白话拆解数据结构(强推)
    • 计组, 操作系统, 计网: 不太好推荐, 我看的是beokayy, 时间短, 精炼, 重点突出, 缺点是需要花钱买。
  • 二轮
    • 王道强化课, b站没有, 看网盘。
    • 数据结构算法题顺序表题目有比快排好用且好写的办法, 好好想想

需要购买的书籍(没有标注就是实体或电子版都可以)

  • (非必须)高教社408大纲, 应该还没有26, 可以买25, 当年新加内容一般当年都不考
  • 408四门王道教材(尽量实体)
  • 408真题
  • (非必须)408各机构模拟题, 一般真题就够用了, 再有时间可以看看还没考过的知识点

其他要说的
- 很多计算机考研学生是跨考的, 或者其他情况, 反正大部分情况下代码水平都堪忧, 什么语法错误, 然后钻牛角尖, 还有什麽申请内存不释放…反正数据结构…我相信大家有能力在数据结构代码题上有自己的想法。不要被其他人误导, 他们说不定还不如你厉害, 有些方法完全就是没必要用, 包括王道讲的一部分代码, 自己好好想想。

数学:

模拟时间安排

  • 12/01 - 12/21: 空着 + 模拟
  • 11/01 - 11/30: 模拟
  • 10/01 - 10/31: 真题 两天一套
  • 09/01 - 09/30: 空着
  • 07/01 - 08/31: 强化
  • none - 06/30: 一轮: 5个月左右(高数3月, 线代1月, 概率论1月)
    • 买个练习册一块做着
    • 建议一轮结束, 复习一下, 然后做做早年真题, 然后根据做题内容回头巩固总结, 早年真题指的是2010之前, 有时间多做点。近年真题2017(数二), 2021(数二)比较简单, 当然也不一定非要做简单的, 反正是建议提早熟悉一下真题

避坑
- b站各种 xxxx哥
- 各种卖课的

学习内容推荐

  • 建议 主+辅 的形式, 且尽量减少视频课时间, 多做题和总结, 太长了你看不下去, 但不是让你不看视频课
    • 主: 考研名师, 主打学习基本知识点, 顺便把每一章节的例题, 课后题, 练习册一块做
      • 我只说一下张宇老师, 张宇老师是一个非常有才华的老师。我说一下优点和缺点: 优点是能帮你拔高, 能帮你从120变成140, 内容多, ; 缺点是你首先得能拔高, 你首先得能到120。张宇老师就是有一种想把他知道的所有的知识都塞给你的感觉, 但是其实你能理解和吸收的就那么多, 他说得太多了, 一战很难抓住重点, 大部分知识时间性价比也不高, 这种情况下他视频课时间还贼长(可能是武的两倍)
    • 辅: 一些小up, 主打跟着练习做题技巧、思路等。还有一些考研名师讲不清楚的地方, 可以听一些b站up的讲解。下面是推荐, b站均能搜到。注意, 每个up每年都有每年的状态, 这些up25考研都很好, 26你们自己看情况
      • 线代铜: 首推, 线代无敌
      • 没咋了(数二), 吃尽天下面(数一): 李林相关内容讲解
      • 你的葫芦_: 张宇相关内容讲解
      • 帕拉迪宇(有时候会改名, 自己找找): 张宇头号黑子, 内容听说很好, 他的观点…希望你们有自己的判断

需要购买的书籍

  • 所跟的老师的教材
  • 各种习题册, 习题册一般不留做题空间, 有平板去微信公众号搜做题本, 没有的话可以去网上买个做题本
  • 历年真题, 且有一点, 早年真题依然有参考价值, 那种年龄比你大的真题也有
  • 模拟题(10月后), 习题推荐看b站up

其他要说的

  • 如果跟的像张宇这种一轮就能把内容讲全的, 一轮结束好好复习一下, 然后做真题
  • 习题册做不了太多也不用慌, 重要的是理解吸收总结, 综合性比较好的像1000题, 好好做完一本就还可以了。880的下位可以做108, 金榜那边可以做330还有武忠祥严选题。基本上完整的做完一本，每个知识点都涉及到就差不多了, 不要把做习题当任务一样, 类似的题和知识点都会了, 确定能做对了就别再一个劲逮着做了, 该跳就跳, 其他

英语(二):

模拟时间安排

  • 后续: 小三门 + 大小作文
    • ”新“题型: 刘琦, 只看方法论(2节, 1h/节), 然后直接做, 挺简单的, 一次可以多做几篇, 数量有限认真对待
    • 翻译: 直接做, 这个熟悉一下流程, 这个做旧题意义就这样, 新题出问题一般都是单词短语问题, 只能说通过旧题训练做题能力
    • 完型: 直接做, 一般问题都不大
    • 作文: 我也不是很懂, 感觉也比较套路化, 建议关注一下热点词的拼写之类的
  • 07/01 - 08/31(暑假期间): 阅读x1/天 + 继续背单词
    • 阅读课(总共30min), b站搜 BV15b421J7Em
      • 反正我看这个够用了, 我自己单词量很一般的情况下, 平均每篇错1个
      • 这个up其他的有需要的看一下吧, 其他的我没看过不做推荐
      • 我感觉数二总体蛮简单的, 出问题了一般都是单词, 长难句理解问题, 缺什么补什么
    • 阅读最终要达到平均 <15min/篇
      • 做完之后复盘, 我觉得没必要精读, 先找出为什麽做错了, 然后让你做错的因素给解决掉, 这个是最重要的, 剩下的看你自己
  • 7月(暑假)之前: 背单词,(至少背完核心词)

政治:

先说避坑

  • 你可能觉得考研政治很难, 实际上除25考研, 其他年份的裸考选择都可能做(30分/满分50）
  • 肖1000不用做, 什麽练习册都不用做
  • 不用开太早, 意义不大
  • 课不用全听, 或者可以一个都不听
  • 不要过度依赖肖4肖8, 或者直接说不要依赖肖4肖8
  • 政治属于及格性质的考试, 大家分数都差不多, 考高考低都很难
  • 考研是有限时间拿尽可能多的分数, 政治的复习内容, 性价比普遍都很低。。。

高性价比过线复习方案

  • 暑假之前不用碰, 可以看看徐涛马原课的讲解, 其余的。。。性价比不高
  • 可以过一遍背诵手册
  • 肖4肖8可以做，但不要过拟合，以做真题的感觉为主，模拟题做多了容易多想
  • 肖4感觉背诵意义不大, 建议背马原通用理论, 其余大题背一下套话, 然后抄材料, 参考内容: b站搜 BV11ykNYiEJc
  • 帕拉迪宇一页纸 + 抄材料
    • 多自己编口诀, 口诀越逆天越好记

政治做题技巧

  • 中肯就选, 绝对一般不选
  • 马原多选不全选, 史纲多选一般不全选
  • 说dang好就选
  • 多选最少选两个, 把错的排了, 把对的选上, 排不掉的也都选上

考研择校

一些重要的话

  • 考虑好了再决定。不要说什么家里让你考，这是你自己的人生，你要对你自己负责，难道你要让你父母为你负责一辈子？
  • 考研是在有限的时间里拿更多的分, 不要说所有的分都要拿到, 不可能
